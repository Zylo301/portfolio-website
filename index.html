Custom Gaming PC Build

Als 13-jähriger Informatiker startete ich mein eigenes PC-Projekt, das sowohl herausfordernd als auch unglaublich lohnend war.

Meine Build-Journey

1. Vorbereitung

Sorgfältige Auswahl aller Komponenten: leistungsstarker Prozessor, ausreichend RAM, schnelle SSD fürs Betriebssystem, zusätzliche HDD für Spiele und eine hochwertige Grafikkarte für anspruchsvolle Anwendungen.

2. Zusammenbau

Systematischer Aufbau: Installation von Netzteil, Prozessor mit Kühler auf dem Mainboard, RAM-Einbau, Mainboard-Montage im Gehäuse, Verkabelung und Installation der Grafikkarte.

3. Optimierung

Erfolgreiche Problemlösung durch BIOS-Anpassungen, Betriebssystem-Setup und Treiber-Installation für einen reibungslosen Betrieb.

Der PC-Bau erweiterte nicht nur mein Hardware-Wissen, sondern verbesserte auch meine technischen Fähigkeiten erheblich. Ich nutze den PC bis heute und bin stolz auf das erreichte Ergebnis.

System-Optimierungs-Tools

Eine Sammlung selbst entwickelter Batch-Skripte zur Automatisierung und Optimierung von Windows-Systemen.

Hauptfunktionen:

  • Automatische Systemreinigung und Wartung
  • Intelligentes Prozess-Management für bessere Performance
  • Automatisierte Software-Installation via Chocolatey
  • Backup-Routinen für wichtige Systemdaten
  • Benutzerfreundliche Menüführung für einfache Bedienung

Verwendete Technologien:

  • Windows Batch-Scripting
  • PowerShell-Integration
  • Windows Registry Management
  • Chocolatey Package Manager
